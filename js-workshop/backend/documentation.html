Restaurant API Documentation

A simple API for managing restaurant menu items and orders

Menu Endpoints

GET/api/menu

Get all menu items

Response

[
  {
    "id": 1,
    "itemName": "Burger",
    "availableQuantity": 20,
    "price": 9.99,
    "createdAt": "2023-01-01T12:00:00.000Z",
    "updatedAt": "2023-01-01T12:00:00.000Z"
  },
  ...
]

GET/api/menu/:id

Get a specific menu item by ID

Parameters

id - Menu item ID (in URL)

Response

{
  "id": 1,
  "itemName": "Burger",
  "availableQuantity": 20,
  "price": 9.99,
  "createdAt": "2023-01-01T12:00:00.000Z",
  "updatedAt": "2023-01-01T12:00:00.000Z"
}

POST/api/menu

Create a new menu item

Request Body

{
  "id": 1,
  "itemName": "Burger",
  "availableQuantity": 20,
  "price": 9.99
}

Response

{
  "id": 1,
  "itemName": "Burger",
  "availableQuantity": 20,
  "price": 9.99,
  "createdAt": "2023-01-01T12:00:00.000Z",
  "updatedAt": "2023-01-01T12:00:00.000Z"
}

PUT/api/menu/:id

Update a menu item

Parameters

id - Menu item ID (in URL)

Request Body

{
  "itemName": "Cheeseburger",
  "availableQuantity": 15,
  "price": 10.99
}

Response

{
  "id": 1,
  "itemName": "Cheeseburger",
  "availableQuantity": 15,
  "price": 10.99,
  "createdAt": "2023-01-01T12:00:00.000Z",
  "updatedAt": "2023-01-01T12:30:00.000Z"
}

DELETE/api/menu/:id

Delete a menu item

Parameters

id - Menu item ID (in URL)

Response

{
  "message": "Menu item deleted"
}
